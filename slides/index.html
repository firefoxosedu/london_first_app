Type of Apps

Hosted VS Packaged

HOSTED

  • web server
  • resources online
  • basic permissions
  • behaves like a bookmark with memory

PACKAGED

  • resources on a .zip file
  • special permissions
  • fully installed on device

Access Level

Web / Privileged / Certified

Web

  • access to web APIs
  • HOSTED apps only get this level

Privileged

  • "privileged" type on manifest
  • special permissions
  • install through trusted origin
  • update through same Market

Certified

  • "certified" type on manifest
  • restricted permissions
  • pre built on device
  • update with FOTA

WebActivities: provider 
navigator.mozSetMessageHandler('activity', function(activityRequest) {
  var option = activityRequest.source;

  if (option.name === "pick") {
    // Do something to handle the activity
    ...

    // Send back the result
    if (picture) {
      activityRequest.postResult(picture);
    } else {
      activityRequest.postError("Unable to provide a picture");
    }
  }
});
WebActivities: consumer 
var activity = new MozActivity({
  name: "pick",
  data: {
    type: "image/jpeg"
  }
});

activity.onsuccess = function() {
  var picture = this.result;
  console.log("A picture has been retrieved");
};

activity.onerror = function() {
  console.log(this.error);
};
