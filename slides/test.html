When is it needed?

  • Access to special APIs
  • Want to install it
  • Communicate with other apps

Type of Apps

Hosted VS Packaged

HOSTED

  • web server
  • resources online
  • basic permissions
  • behaves like a bookmark with memory

PACKAGED

  • resources on a .zip file
  • special permissions
  • fully installed on device

Access Level

Web / Privileged / Certified

Web

  • access to web APIs

Privileged

  • "privileged" type on manifest
  • special permissions
  • install through trusted origin
  • update through same Market

Certified

  • "certified" type on manifest
  • restricted permissions
  • pre built on device
  • update with FOTA
